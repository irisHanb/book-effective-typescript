# 동적 데이터에 인덱스 시그니처 사용하기

- 동적 데이터, key 의 타입과 value 의 타입이 동일한 자료 형태에 유용하게 사용할 수 있다. 
  - key 의 타입은 string, number, symbol, Template literal 타입만 가능
  - 존재하지 않는 속성에 접근할 수 있으므로 주의 ( 값타입에 undefined 를 추가해 주자)
  - 동일한 key 를 가질 수 없다.
- 가능한한 정확한 타입을 사용하자.
